JavaScript

JavaScript는 웹페이지에 상호작용성과 특별한 효과를 제공하는 주요한 웹 기술입니다.

만약 아주 기본적인 웹사이트보다는 좀 더 나은 것을 구축하길 원하신다면, 여러분은 몇 가지 기본적인 스크립팅을 배워야 합니다. 기본을 익히는 것은 어렵지 않지만, JavaScript는 많은 복잡한 특징을 가진 강력한 기술입니다. 그렇기 때문에 올바른 학습 방법이 하나만 있는 것은 아닙니다. 우리는 여러분이 어떤 기술과 지식을 익힐 수 있도록 다음의 페이지들로 시작하기를 제안합니다. 첫 번째 기본 기술로부터 고급 기술로 이동하거나, 여러분에게 흥미로워보이는 페이지를 골라보세요!
